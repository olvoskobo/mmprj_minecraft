# mmprj_minecraft
Yandex Practicum CV Sprint1

Проект по детекции персонажей майнкрафт - учебный проект Yandex Practicum CV Sprint1 содержит onedl_mmprj_minecraft_final.ipynb, в котором
1. преобразование датасета из формата PASCAL в формат для YOLO и MMDetection,
2. EDA и перераспределение обучающей-тестовой выборок (в исходном разбиении в валидационной выборке не было нескольких классов, в обучающей один из важных классов был представлен 2 объектами)
3. Обучение модели в MMdetection (отдельный блокнот с результатами запуска на ВМ)
4. Обучение модели в YOLO (отдельный блокнот с результатами запуска на ВМ)
5. Инференс моделей на изображениях и видео
6. Графики по логу обучения в MMDetection и YOLO
7. PDF-отчет
8. Директории с результатами
